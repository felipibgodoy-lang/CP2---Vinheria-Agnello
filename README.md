# 🍷 Vinharia Agnello – Checkpoint 02

Sistema de avaliação de estoque e tendência de vinhos, desenvolvido como parte do Checkpoint 02 da disciplina de Web Development da FIAP.

---

## 📄 Descrição

Este projeto dá continuidade à Vinharia Agnello (CP01), evoluindo o sistema de cadastro de vinhos com foco nos fundamentos do JavaScript:

- Cadastro de **vários vinhos em sequência** via `prompt()` e `alert()`
- **Funções reutilizáveis** para validação de entradas, verificação de estoque e classificação por safra
- **Laços de repetição** (`while`) para controle do fluxo de cadastro e revalidação de dados
- **Escopo de variáveis** respeitado — variáveis locais dentro das funções
- Armazenamento de dados em **variáveis individuais** (`nome1`, `nome2`... sem uso de arrays ou objetos)
- **Relatório final** com total de cadastros, vinhos com estoque baixo e o vinho com a safra mais antiga
